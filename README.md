# Professional Resume

A responsive, browser-based resume built using modern web technologies.

This project demonstrates how a professional resume can be presented as an interactive website instead of a traditional PDF. Throughout development, multiple versions were created to refine the design, improve usability, and experiment with different layouts and styling techniques.

## Features

- 📄 Professional resume presented as a website
- 💻 Responsive layout for desktop and mobile devices
- 🎨 Modern, clean user interface
- ⚡ Fast loading with lightweight design
- 📱 Mobile-friendly experience
- 🧩 Multiple development iterations

## Project Structure


## Getting Started

### Clone the repository

```bash
git clone https://github.com/endritngjelina/Professional-Resume.git
```

### Run locally

Simply open the latest HTML file in your browser, or use a local web server for development.

Example using Python:

```bash
python -m http.server
```

Then visit:

```
http://localhost:8000
```

Or use the **Live Server** extension in Visual Studio Code.
